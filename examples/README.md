# Example request

The repository intentionally does not bundle a real student's manuscript or a protected university template.

Use a synthetic or fully anonymized test document when evaluating the workflow:

```text
Use $format-thesis-docx to format my uploaded DOCX using the official guide and
approved sample. Preserve all academic content, equations, citations, tables and
figures. Create a new file, normalize Heading 1–3 and multilevel numbering,
configure front-matter and body page numbering, insert a dynamic Word TOC, and
ask only about consequential conflicts between the guide and sample.
```

Expected deliverables:

- a new formatted `.docx`;
- a concise change summary;
- recorded consequential confirmations and assumptions;
- any remaining Word field-update or rendering limitation.

See [`docs/validation.md`](../docs/validation.md) for the latest demonstrated verification scope.
